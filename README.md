# Bakery Database System

## Overview
This project implements a relational database for a bakery using PostgreSQL.
It models sales, customers, employees, products, and inventory based on a
20,000-transaction dataset.

## Features
- Normalized relational schema
- Analytical SQL queries
- Indexing for performance testing
- Cloud-hosted PostgreSQL (Neon)

## Project Structure
create_tables/ -- table schemas and constraints
analysis_queries/ -- analytical SQL queries
index_test/ -- index performance tests

## Technologies
